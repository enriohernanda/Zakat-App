# ZakatApp

Aplikasi manajemen zakat berbasis web yang modern untuk membantu pengguna menghitung, mencatat, dan memantau zakat dengan mudah dan efisien.

🔗 Live Demo: https://zakat-app-phi.vercel.app/

## Fitur Utama

- Autentikasi (Login & Register)
- Dashboard dengan statistik & ringkasan data
- Kalkulator Zakat (Zakat Maal & Profesi)
- Riwayat transaksi zakat
- Fully Responsive (Mobile, Tablet, Desktop)
- UI modern (fintech / SaaS style)
- Performa cepat dengan Next.js

---

## Tech Stack

- **Frontend:** Next.js 16, React 19, TypeScript  
- **Styling:** Tailwind CSS, Shadcn UI  
- **Backend:** Next.js API Routes  
- **Database:** Prisma ORM + SQLite  
- **Autentikasi:** NextAuth.js, JWT, bcrypt  
- **Validasi Form:** React Hook Form + Zod  
- **Visualisasi Data:** Recharts  
- **Deployment:** Vercel
